Export Trimmed Layer to PNG

This script allows you to export the currently selected layer in Krita as a trimmed PNG file. The export dialog window opens, and new image is then exported as a PNG file.

Usage

To use this script:

  1. Open Krita and load your document.
  2. Select the layer you want to export.
  3. Click on Tools → Scripts → Export single layer.
  4. Choose the location and name for the exported PNG file when prompted.
